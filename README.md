# Ballieue

## Overview

**Ballieue** is an innovative mobile app designed to motivate kids to play soccer outside and get their daily exercise by making it more interactive and fun. The app aims to enhance the social skills of children aged 12-15 by encouraging them to participate in public soccer games at local Cruyff Courts.

## Features

- **Skill Cards**: Each child has their own card displaying their level in various soccer skills.
- **Public Games**: Kids join public games and play soccer against other kids at Cruyff Courts.
- **Leveling Up**: By participating in games, kids can level up their skill cards, reflecting their progress and achievements.

## Technologies Used

Ballieue is built using the following technologies:

- **Front-end**: React Native with Expo
- **Back-end**: Appwrite

## Installation

To run Ballieue locally on your machine, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/issamfissoune/Ballieue.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ballieue
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   expo start
   ```
